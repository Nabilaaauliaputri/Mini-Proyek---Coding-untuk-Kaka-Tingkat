## Penjelasan Program 
-Bahasa pemograman yang digunakan: C
-Fitur utama:
 -Pengingat belajar otomatis selama sesi belajar.
 -Jeda waktu antar pengingat menggunakan fungsi khusus.
 -Perulangan untuk menampilkan pesan pengingat berkali-kali.
 -Pesan motivasi yang muncul secara berkala.
 -Pesan penutup setelah sesi belajar selesai.

-Konsep yang digunakan:
-Input/output (printf untuk menampilkan pesan)
-Variabel (menyimpan durasi, interval, jumlah pengingat)
-Perulangan (loop) (for untuk mengulang pengingat) (while untuk menunggu jeda sampai waktu yang diinginkan tercapai)
-Fungsi (fungsi clock untuk membuat jeda waktu)
-Library (<stdio.h> dan <time.h>)
